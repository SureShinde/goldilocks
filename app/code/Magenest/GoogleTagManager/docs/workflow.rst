Workflow
========

This module uses change-log driver releases. This means that publishing new releases for
the module is done by the Build Server (CI, Jenkins) and developer just expresses their
intent on creating a new release by adding a new record to the change-log.

The guides on how to work with this module can be found `HERE <https://docs.Magenest.com/workflow-module--v2/guides.html>`__.

When using this module as a dependency, consider the following `HINTS <https://docs.Magenest.com/workflow-module--v2/guides.html#dependencies>`__.
